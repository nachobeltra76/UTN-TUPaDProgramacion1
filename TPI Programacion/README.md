Este proyecto forma parte del trabajo práctico final de Programación en Python, y tiene como objetivo analizar y manipular información sobre distintos países a partir de una lista o base de datos.
Permite realizar operaciones como filtrado, conteo, comparación y ordenamiento de países según diferentes criterios (continente, población, superficie, etc.).

El programa está estructurado de forma modular, separando la lógica principal (Main.py) de las funciones auxiliares (Funciones.py), para favorecer la claridad y la reutilización del código.
